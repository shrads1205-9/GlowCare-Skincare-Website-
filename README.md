# GlowCare - Skincare Website

GlowCare is a simple and responsive skincare product discovery website developed using HTML and CSS.

## Features

- Responsive navigation bar
- Hero section
- Product categories
- Featured skincare products
- Popular skincare brands
- Customer reviews
- Daily skincare routine section
- Responsive design

## Technologies Used

- HTML5
- CSS3

## Project Structure

```
GlowCare
│
├── index.html
├── style.css
└── images
    ├── serum.jpg
    ├── sunscreen.jpg
    ├── moisturizer.jpg
    └── facewash.jpg
```

## Purpose

This project was created as a beginner web development project to practice HTML and CSS concepts and build a simple product showcase website.
